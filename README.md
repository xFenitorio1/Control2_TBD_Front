# TBD Control 2 - Frontend

Este proyecto es la interfaz de usuario para el Control 2 de Taller de Bases de Datos (TBD), desarrollado con Vue 3 y Vite.

## Requisitos Previos

Asegúrate de tener instalados:
- [Node.js](https://nodejs.org/) (versión 16+ recomendada)
- npm (normalmente viene instalado con Node.js)

## Instalación

1.  Clona este repositorio o descarga los archivos.
2.  Abre una terminal en la carpeta raíz del proyecto (`.../Control 2`).
3.  Instala las dependencias ejecutando:

```bash
npm install
```

## Ejecución en Desarrollo

Para iniciar el servidor de desarrollo y ver la aplicación en tu navegador:

```bash
npm run dev
```

Por defecto, la aplicación estará disponible en `http://localhost:5173/` (o el puerto que indique la terminal).

## Construcción para Producción

Para compilar la aplicación para producción (generará la carpeta `dist`):

```bash
npm run build
```

## Tecnologías Utilizadas

-   **Framework:** Vue 3
-   **Build Tool:** Vite
-   **Estado:** Pinia
-   **UI:** Vuetify 3
-   **Mapas:** Leaflet / Vue-Leaflet
-   **HTTP Client:** Axios
